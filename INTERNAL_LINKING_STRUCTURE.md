## Internal Linking Strategy

### Hub-to-Spoke Linking
Each hub page links to:
- All of its spoke pages (as "Related Resources")
- Other core hub pages (peer hubs)

### Spoke-to-Hub Linking
Each spoke page links to:
- Its parent hub page
- 2-3 peer spoke pages (other spokes under the same hub)

### Outer-to-Core Linking (Authority Building вҶ’ Monetization)
Each outer cluster topic page links to:
- Relevant core hub pages (the primary link flow)
- Relevant core spoke pages

Examples:
- `/injury-prevention/elbow-pain/` вҶ’ `/deadhang-guide/` + `/deadhang-safety/`
- `/complementary-exercises/pullup-training/` вҶ’ `/training-programs/`
- `/equipment-reviews/budget-bars/` вҶ’ `/equipment-guide/` + `/best-pullup-bars/`

### Homepage Linking
The homepage features:
- Prominent navigation to all 6 core hubs
- Feature grid linking to all core hubs
- Secondary feature grid linking to all outer clusters
